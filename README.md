# FeelMe

## NOTE: Script

- ### Init project :

```bash
npm install
```

- ### Start development :

```bash
npm run dev
```

- ### Run test :

```bash
npm run test
```

- ### Run test with coverage :

```bash
npm run coverage
```

- ### Run test with Vite-UI :

```bash
npm run ui
```

## NOTE: Command Option

```bash
--silent    #for silent output
--ui        #for start and open vite-ui
```
